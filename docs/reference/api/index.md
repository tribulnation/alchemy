# Overview

The API reference is generated from the Python package surface.

## Client

- [`Alchemy`](alchemy.md): root constructor and API group accessors.

## API Groups

- [`nft`](nft.md): NFT API methods exposed through `client.nft(network)`.
- [`portfolio`](portfolio/index.md): Portfolio API methods exposed through `client.portfolio`.
- [`prices`](prices.md): Prices API methods exposed through `client.prices`.
- [`token`](token.md): Token JSON-RPC methods exposed through `client.token(network)`.
- [`transfers`](transfers.md): Transfers JSON-RPC methods exposed through `client.transfers(network)`.
- [`utility`](utility.md): Utility JSON-RPC methods exposed through `client.utility(network)`.
- [`simulation`](simulation.md): Simulation JSON-RPC methods exposed through `client.simulation(network)`.
